Everything your household needs

Built for real families — from first-time radio operators to licensed net control stations.

🗣️
Live transcription

Incoming voice hits text in real time via Whisper — partials appear while the operator is still talking.

👥
Multi-user profiles

Each family member has their own login, avatar, voice, and preferences. No shared account juggling.

🎙️
Voice PTT

Speak into your browser mic. The server transcribes your voice, synthesizes it with Piper TTS, then broadcasts it over the air — callsign included.

♿
Senior-friendly design

WCAG 2.1 AA, large touch targets, Read Aloud mode, and plain-language status messages — no jargon.

🌦️
SKYWARN weather alerts

National Weather Service alerts for your county push to every connected device and auto-announce over the air.

📔
Session journals

Save and publish transcripts from each radio session. Optional AI summaries powered by Gemini.

📞
FCC callsign lookup

Verify GMRS and amateur callsigns against the FCC database. Contacts show a green ✓ badge when confirmed.

🐳
One-command install

Docker Compose on any Linux server. No database to manage — flat-file storage keeps it simple and crash-resistant.

📊
Real-time spectrogram

A live waterfall display shows radio activity with squelch and voice activity indicators — switch between viridis and grayscale colormaps.

Extend it for your net

Radio-TTY ships with two plugins built in. More can be added without touching the core server.

🎛️ Net Control Station Built-in

Full-featured NCS panel for managing radio nets — roster, emergency interrupt, and audio replay.

  • ✦
    Live roster
    Check operators in and out with traffic priority levels — Routine through Emergency.
  • ✦
    BREAK BREAK interrupt
    Drains the TX queue and broadcasts an emergency interrupt to clear the channel instantly.
  • ✦
    15-second audio replay
    Replay the last 15 seconds of received audio when you need to catch a missed transmission.
  • ✦
    Auto net ID
    Periodic station announcements at a configurable interval — FCC-compliant by default.
🌩️ SKYWARN Alerts Built-in

Monitors your NWS county zone every 5 minutes and pushes severe weather alerts to the whole household.

  • ✦
    Red alert banner
    Active NWS warnings appear at the top of every connected browser immediately.
  • ✦
    Browser notifications
    Opt-in push notifications reach family members on background tabs or locked screens.
  • ✦
    Auto-announce over the air
    When the channel is clear, alerts are automatically read aloud over GMRS via TTS.
  • ✦
    Per-county targeting
    Set your NWS zone code once in config — only alerts relevant to your area come through.
🔌
Build your own plugin

Plugins hook into five events: on_rx_final, on_audio_rx_chunk, on_audio_tx_pre_queue, and more. Subclass BasePlugin and drop it in — no core changes needed.

Up and running in minutes

Radio-TTY runs anywhere Docker runs — any Linux server on your home network. No cloud accounts, no monthly fees.

Quick install
# 1. Clone the repo
git clone https://github.com/Xpiatio/Radio-TTY
cd Radio-TTY

# 2. Run setup (creates .env)
./setup.sh

# 3. Start the stack
docker compose up -d

# 4. Open in your browser
http://your-server-ip

What you need
🖥️
Any Linux server
x86, ARM, old laptop, NAS — if it runs Docker, it runs Radio-TTY.
🎙️
USB audio interface
A USB sound card connected to your GMRS radio's speaker and mic jacks.
📻
GMRS radio + license
A Part 95-certified GMRS radio and an FCC GMRS license (covers your whole family).
🌐
Local network
Family members connect from any device on your home Wi-Fi — no port forwarding needed.
